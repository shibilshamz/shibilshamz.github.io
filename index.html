Stock Trading Agent v2

Live · 24/7 VPS · NSE Markets
Python Groq LLaMA 3.1 Telegram Bot Ubuntu VPS Supervisor MCP Server pandas-ta

An autonomous AI agent that scans the top 20 Nifty 50 stocks every 15 minutes during market hours. It scores each stock across three technical signal layers (ORB, VWAP, Momentum), validates the composite score with a Groq LLaMA model against live news sentiment, and executes paper trades with dynamic ATR-based position sizing. Every signal, entry, exit, and end-of-day P&L is pushed to Telegram in real time.

Key results

  • Agent has run continuously for 6+ months with zero manual restarts
  • 3-layer signal system reduces false-positives vs. single-indicator approaches
  • Custom MCP server with 7 tools enables natural-language control via Claude Desktop
